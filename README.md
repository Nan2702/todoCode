# 🛒 TodoCode - Supermercado

Proyecto de ejemplo desarrollado en **Java + Spring Boot**, con integración de **Docker Compose** y base de datos **MySQL**.  
El sistema gestiona productos, sucursales y ventas de un supermercado.

---

## 🚀 Tecnologías utilizadas
- Java 17
- Spring Boot
- Maven
- MySQL
- Docker / Docker Compose
